**Hello Astha** is a modern, responsive social media application designed to connect people.[[4](https://www.google.com/url?sa=E&q=https%3A%2F%2Fvertexaisearch.cloud.google.com%2Fgrounding-api-redirect%2FAUZIYQHSR1h3VnPx-4syFRhbqxGcrVLCt8LM1giSikzqx4cGIFxTiXtAp3ed4cF1otqr1cRy7WKkpXsnfpREdHofH9_g0j1w8lW987XpBkeNeoZbhTpI6LQo-llpHhudEyJOUPHAMm1MCg%3D%3D)] Built with the MERN stack, it features real-time updates, secure authentication, and a sleek Facebook-inspired UI.

## ✨ Features
- 🔐 **Secure Auth:** Sign up and Login using JWT and Bcrypt.
- 📰 **Dynamic Feed:** View posts from people you follow.
- 📸 **Multimedia Posts:** Share text updates and images.
- ❤️ **Interactions:** Like and comment on your friends' posts.
- 👤 **User Profiles:** Customize your bio and profile picture.[[5](https://www.google.com/url?sa=E&q=https%3A%2F%2Fvertexaisearch.cloud.google.com%2Fgrounding-api-redirect%2FAUZIYQErZw_6FJCP_8cdmqZIh9HvDU2iKVe6C-PcdZtK7y8qrryfCfkgqwg3YVDZ3Wc-PjLNGgzB2err4w007MWAHBeU5WD5LGMev-ecdxbzNkKXQAhJOQyoUv4dL4I9k-4gdyN31jHC0JLmbu4_cOicyXVlaabGpz2QYvaklMTIvsOP7kfE55wvKagWBeg%3D)]
- 📱 **Fully Responsive:** Optimized for both Desktop and Mobile.
